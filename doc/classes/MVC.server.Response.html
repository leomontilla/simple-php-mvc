redirect()

redirect(string $url, integer $status)

Redirect

This method immediately redirects to a new URL. By default, this issues a 302 Found response; this is considered the default generic redirect response. You may also specify another valid 3xx status code if you want. This method will automatically set the HTTP Location header for you using the URL parameter.

Parameters

string $url

The destination URL
integer $status

The HTTP redirect status code (optional)

render()

render(mixed $response, integer $status) : void

Renders a response.

Parameters

mixed $response

The response to be rendered. Can be an array

                      containing 'body', 'headers', and/or 'status'
                      keys, or a string which will be used as the
                      body of the response.  Note that the headers
                      must be well-formed HTTP headers, and the
                      status must be an integer (i.e., the one
                      associated with the HTTP status code).  The
                      response body is chunked according to the
                      buffer_size set in the constructor.
integer $status

The status of the response
